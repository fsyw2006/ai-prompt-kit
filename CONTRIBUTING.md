# Contributing

Thanks for helping improve AI Prompt Kit.

## Good First Contributions

- Improve README examples
- Add prompt export formats
- Add tests for CLI behavior
- Improve error messages
- Add prompt templates for common workflows

## Development

Run the CLI from the project root:

```bash
python -m aipromptkit --help
```

Run tests:

```bash
python -m unittest
```

## Pull Request Guidelines

- Keep changes focused
- Add or update tests for behavior changes
- Update documentation when user-facing commands change
- Prefer clear error messages over silent failures
