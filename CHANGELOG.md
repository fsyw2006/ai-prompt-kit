# Changelog

## 0.2.0 - Unreleased

- Added automatic tag inference with `add --auto-tags`.
- Added prompt quality scoring with the new `quality` command.
- Added JSON and CSV import support.
- Improved fuzzy search and Unicode-aware keyword search.
- Improved data-file validation for corrupted JSON and invalid schemas.
- Added atomic writes for prompt data saves.
- Rejected empty tags and non-positive prompt ids.
- Added tests for CLI behavior, imports, corrupted data, fuzzy search, and quality scoring.
- Reworked README, contributing guidance, roadmap, and CI smoke checks.

## 0.1.0

- Initial CLI for saving, listing, searching, copying, and exporting prompts.
- Added prompt variables, Markdown import, backups, usage statistics, tags, and starred prompts.
- Added project documentation, issue templates, and contribution guidelines.
